# Spotify PWA

App de música

### Dependencias

Depois de clonar o repositório, é necessário instalar as dependências.

Node version: V20.18.1

Execute:

```bash
yarn
```

### Environment Variables

Criar um arquivo .env.local com as variaveis que está no .env.example

## Run

Execute:

```bash
yarn dev
```
